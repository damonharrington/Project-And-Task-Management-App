# Project-And-Task-Management-App
This desktop-based GUI application, developed using Python and Tkinter, provides an intuitive and user-friendly interface for handling these essential project management activities.

The application empowers users to create and organize tasks, specifying details such as the title, description, group affiliation, responsible team member, priority level, and deadline. Users can define multiple groups and assign members to each group, making it easier to distribute tasks based on team structure or departmental divisions. This supports a more organized workflow and helps in managing workloads effectively.
